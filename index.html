SUPPORT

Common Questions

Everything you need to know about REDitors.

Is there a free version?

Yes, REDitors offers a robust Free plan that includes all essential editing tools and 720p export. No watermarks for personal use.

Does it work on Windows and Mac?

Absolutely. REDitors is fully optimized for both Windows 11 and macOS Sonoma, including native Apple Silicon support.

Can I import my existing projects?

We support XML and EDL import from Premiere Pro, DaVinci Resolve, and Final Cut Pro so you can switch seamlessly.

What are the system requirements?

We recommend 16GB RAM and 4GB VRAM for 4K editing. However, our proxy workflow allows editing on lower-end systems.
